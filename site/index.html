SIMULABRA: INFINITE SOFTWARE

Simulabra is an object-oriented programming system for the Web. It's not just a programming language though: it's everything you need to make almost any kind of software, all in one place, with the help of ubiquitous artificial assistance.

Language

A simple, uniform syntax and a fresh take on object-oriented programming make an ideal medium for sculpting ideas.

Editor

A truly integrated Web-based development environment built from the ground up with AI and malleability in mind.

Platform

Take the agony out of actually running software with native deployments, distributed computing, and storage, whether local, self-hosted, or in the cloud.

Simulabra is currently in early alpha and many parts are incomplete. Check out the source for more information.
