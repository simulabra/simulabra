Simulabra

the software construction kit

Simulabra is an object-oriented programming system for the Web. It has everything you need to make almost any kind of software with the help of ubiquitous artificial assistance. Design, build, test, and run all in one place, whether for an AI cooking assistant, Discord bot, or multiplayer game.

it's a language

A simple, uniform syntax and a fresh take on object-oriented programming make an ideal medium for sculpting ideas.

an editor

Web-based IDE built from the ground up with AI and malleability in mind, for beginners and experts alike.

and a platform

Focus on building and take the agony out of shipping software whether local, self-hosted, or in the cloud.

Simulabra is currently in early alpha and many parts are incomplete. Check out the source for more information.
