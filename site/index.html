Simulabra

Simulabra is an object-oriented programming system for the Web. It isn't just a programming language though.

Language

A simple, uniform syntax and a fresh take on object-oriented programming make an ideal medium for sculpting ideas

Editor

A truly integrated Web-based development environment built from the ground up with AI and malleability in mind

Platform

Take the agony out of actually running software with native deployments, distributed computing, and storage, whether local, self-hosted, or in our cloud

Simulabra is currently in early alpha and many parts are incomplete. Check out the source for more information.
